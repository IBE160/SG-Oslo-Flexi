# Brainstorm – UI/UX Design (2025-11-09) — MVP aligned

## Screens & flow
1) **Upload**
   - Drag-and-drop + file picker
   - Helper text (MVP): Formats = PDF/DOCX/PPTX/TXT/MD · English only · ≤ 20 MB
   - Primary: **Upload**
2) **Processing**
   - Stepper: Upload → Processing → Quiz → Results
   - Inline status: “Parsing text…” / “Running OCR (if needed)…” / “Generating quiz…”
3) **Quiz**
   - 5 MCQs, radio options, “Submit”
   - Accessibility: keyboard nav (Tab/Arrow), labels/titles
4) **Results**
   - Score out of 5
   - Per-item feedback (Correct/Incorrect), show correct option
   - Link: “Start over” (clears session data)
